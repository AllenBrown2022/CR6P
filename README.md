# 六人格博士论文审稿 Prompt

> 用一份提示词，把一篇博士论文交给六位脾气、读法、目标各不相同的"审稿人"过一遍。
> A six-persona PhD-dissertation peer-review prompt for chat LLMs (Chinese).

## 这是什么

一份可直接粘贴使用的**系统级提示词**。把它喂给任意一款对话式大模型（Claude、GPT 等），再附上你的论文，它会分别以**六种审稿人人格**评审你的博士学位论文：给出分档、学术评语、按 `P0/P1/P2` 分级的修改建议，以及——每个人格专属的**防御性写作建议**（教你怎么提前堵住这一类审稿人会挑的刺）。

它**不内置**任何具体论文内容，也**不内置**某一所学校的评阅书模板：论文与评分表都由你在运行时提供。人格框架固定，**学科与领域可替换**。

## 它解决什么问题

送审/答辩前，你其实想知道的是："不同类型的审稿人，会从哪儿开始怀疑我？" 一个急脾气的大同行 20 分钟扫完图表就直奔结论，一个老学究花一下午逐字抠你的符号，一个邻近领域的善意读者只想找到能写进推荐信的亮点——他们的雷区完全不同。这份提示词把这几种**真实存在的读法**显式化，让你在交稿前自己先走一遍，把"会被问到的问题"提前改掉。

## 六种人格一览

| # | 人格 | 画像 | 读法 | 重点盯什么 |
|---|------|------|------|-----------|
| 一 | 效率扫描型 | 大同行 · ~20 分钟 | 倒序速读、跳着看 | overclaim、图表自明性、跨章导航成本 |
| 二 | 严谨学究型 | 老学究 · 一下午 | 顺序逐字 | notation 零容忍、详略得当、跨领域"串线"、伏笔兑现、导游式承上启下 |
| 三 | 学习交流型 | 非小同行 · 1–2 天 | 善意通读 | `unique` + `challenging` 两条金标准、可写进推荐信的亮点 |
| 四 | 高效严格型（一×二） | — | 先扫靶子，再逐字钉死 | 数值一致性系统核查、投入产出比 |
| 五 | 友好高效型（一×三） | — | 快翻 + 善意 | 落地路径、答辩准备（最温暖） |
| 六 | 严格的伯乐（二×三） | — | 逐字挑错，却为推荐而读 | 符号零容忍 + 主动找亮点 + 把局限重构为"未来接口" |

> 人格一/二/三是三种独立的基础读法；四/五/六是它们的两两组合（派生人格），行为应是对应两个基础人格的逻辑叠加。

## 快速开始

1. **复制** 主提示词文件（`PROMPT.md`）全文，作为系统提示 / 第一条指令，贴进你常用的对话式 LLM。
2. **附材料**：待审论文（PDF 或正文文本，含图、表、公式）；可选地附上你所在单位的评阅书模板（不提供时走内置通用占位模板）。
3. **发指令**，例如 `请生成六人格完整评审`。

### 触发指令

| 指令 | 效果 |
|------|------|
| `请用【人格X】审阅以下论文` | 只输出该单一人格 |
| `请用【人格四 / 五 / 六】审阅` | 输出对应组合人格 |
| `请生成六人格完整评审` | 依次输出六个人格 + 综合裁决 |

### 可选调用参数

`每人格评语 ≤ 200 字` ｜ `只给 P0` ｜ `评分用表格` ｜ `按答辩前最后一轮把关从严` …… 直接附在指令后即可。

## 输出长什么样

每个人格固定按五段输出：① 一句话总印象 → ② 评分表（含分档、是否同意答辩、熟悉程度）→ ③ 学术评语（散文体，带该人格语气）→ ④ 不足与建议（`P0` 必改 / `P1` 建议改 / `P2` 可选）→ ⑤ 防御性写作建议。提示词内置了人格一、二、四的完整 **Few-shot 示例**与一组"违反铁律的反例"，模型据此对齐格式与语气。

## 反幻觉设计

LLM 写评审最大的风险是**一本正经地编造**（虚构数值、臆造图表、拿匿名化缺失扣分）。提示词把一组**反幻觉铁律**设为最高优先级，凌驾于一切人格之上：

- 一切事实性判断只以论文真实呈现的内容为依据，凡引用须能定位到章节/图/表号；
- 不得编造数值、公式、参考文献、页码、章节结论；
- 严格区分"论文确实存在的硬伤"（可影响分档）与"审稿人想看到但非必需的内容"（只能写成建议）；
- 匿名送审被隐去的信息（署名、机构、伦理号、成果清单等）一律不计入扣分；
- 不确定时降低断言强度，而非编造确定性结论。

并附"反例区块"，把三类典型幻觉直接示范为"禁止这样写"。

## 适配你自己的学科 / 评分表

- **换学科**：提示词里的学科是占位符 `〈论文所属一级/二级学科〉`，填入论文实际所属学科即可；六人格本身与领域无关。
- **换评分表**：调用时附上你所在单位的真实评阅书模板（维度名称、各档位定义），覆盖内置默认。各校"评阅书/评议书"的维度并不统一，内置的一套只是常见占位。
- **换严格度**：声明场景（盲审 / 预答辩自查 / 投稿前自审 / 答辩前最后一轮），人格不变、把关力度按需收紧或放松。

## 设计说明与出处

- **基础人格（一 / 二 / 三）** 的要素来自 **林丰涵老师** 在 Bilibili 的分享，由本项目消化归纳为结构化描述。
  > 原视频：[盲审专家怎么看博士论文](https://www.bilibili.com/video/BV1stFxzFETt)
- **复合人格（四 / 五 / 六）的组合方式、整份提示词的工程化、反幻觉铁律与全部 Few-shot 示例，均为本项目原创。**
- 提示词**不含**任何具体论文、个人或机构信息；所有示例均为占位/虚构，仅用于演示格式与反应方式。
- 六个人格是一种**思维脚手架**，是对"几类常见读法"的归纳，而非对任何真实审稿人的精确还原。

## 局限与免责

- 大模型的输出**仍可能含错误或幻觉**，务必人工复核，**不可直接**作为正式评审意见提交。
- 输出的分档为**模拟结果**，仅供作者自查参考，**不代表**任何机构或任何真实评审人的判断。
- 本工具定位是**"写作自查 + 答辩预判"**，不替代真实的同行评议或导师把关。
- 不同模型、不同版本的表现会有差异；提示词不保证在所有模型上行为一致。
